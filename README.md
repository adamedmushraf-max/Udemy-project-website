# Udemy-project-website


A simple front-end Udemy-inspired website built using HTML and CSS. This project displays available courses in a clean and responsive layout, replicating the basic design and structure of an online learning platform.

## 🚀 Features

- Responsive navigation bar
- Course listing section
- Course cards with images and details
- Clean and modern UI design
- Fully built with HTML and CSS
- Mobile-friendly layout

## 🛠️ Technologies Used

- HTML
- CSS

## 📂 Project Structure

```
Udemy-Clone/
│
├── index.html
├── style.css
├── images/
└── README.md
```

## 🎯 Purpose

This project was created to practice and improve front-end web development skills, including:

- HTML page structure
- CSS styling and layouts
- Flexbox positioning
- Responsive web design
- UI replication from real-world websites


## 📖 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/udemy-clone.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser.
